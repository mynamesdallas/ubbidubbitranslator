What is Ubbi Dubbi?

Ubbi Dubbi is a playful, secret language game where you insert 'ub' before every vowel sound in a word, popularized by the kids' show ZOOM in the 1970's, making it sound like 'Hubi Frubiends' for 'Hi friends'.

Learn the language rules

  • Find the vowel: You need to identify each vowel in the word (a, e, i, o, u).
  • Add "ub": Place the syllable "ub" before every vowel in the word.
  • Stress the "ub": Make sure to put emphasis on the "ub".
  • Use simple phrases: Learn common phrases like "Gubive mube uba hubigh fubive" to get a good idea of the language.
  • Make it a game: Practice speaking the language with friends and family and throw off everyone around that doesn't know it.
